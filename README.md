# Aotoki

Nothing in this gem, just a practice.

## Installation

Add this line to your application's Gemfile:

    gem 'aotoki'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install aotoki


## Usage

Current only support run `aotoki` command.
